# 🎬 Movie Recommendation System

This project is a **Content-Based Movie Recommendation System** built using **Machine Learning and NLP techniques**.

---

## 🚀 Features

- Recommends movies based on similarity
- Uses movie metadata (genre, cast, keywords, overview)
- Fast and efficient recommendation system

---

## 🧠 Technologies Used

- Python
- Pandas
- Scikit-learn
- NLP (CountVectorizer)
- Cosine Similarity

---

## ⚙️ How It Works

1. Load movie and credits datasets  
2. Merge datasets  
3. Extract important features  
4. Convert text data into numerical vectors  
5. Calculate similarity between movies  
6. Recommend top 5 similar movies
